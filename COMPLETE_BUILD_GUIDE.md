## ğŸ› TROUBLESHOOTING

### Build Won't Start

**Problem:** "Build failed" or "No credentials"

**Solution:**
```bash
eas credentials
eas build --platform android --profile production
```

### APK Won't Download

**Problem:** Download link not appearing

**Solution:**
```bash
eas build:list
```

This shows all your builds with download links.

### Installation Fails

**Problem:** "App not installed"

**Solution:**
- Uninstall any old version first
- Enable "Unknown sources" in phone settings
- Try transferring via USB instead
- Restart phone and try again

### App Crashes on Launch

**Problem:** App opens then closes

**Solution:**
1. Grant all permissions
2. Check phone has internet
3. Clear app data and reinstall
4. Check Android version is 8.0+
