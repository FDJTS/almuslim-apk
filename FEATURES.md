
### 2. Update app.json for notifications
```json
{
  "plugins": [
    ["expo-notifications", {
      "sounds": ["./assets/sounds/adhan.mp3"]
    }]
  ],
  "permissions": [
    "NOTIFICATIONS",
    "CAMERA",
    "LOCATION"
```

Create `assets/sounds/` directory and add:
